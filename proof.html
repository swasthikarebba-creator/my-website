Proof of 1=2

a=b [1]
/* multiply both sides by a */
a*a = a*b
a^2 = a*b
/* subtract by b^2 both sides */
a^2 – b^2 = ab – b^2
(a-b) (a+b) = b(a-b)
/* Cancel (a-b) both sides */
a+b = b
/* substitute a=b from [1]*/
b+b = b
2b = b
2=1
